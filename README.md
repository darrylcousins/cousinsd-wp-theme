# cousinsd-wp-theme

A simple Wordpress theme


## Install

```shell
git clone git@github.com:darrylcousins/cousinsd-wp-theme.git
cd cousinsd-wp-theme
```

Install packages:

```shell
npm install
```

Serve development using `gulp watch` and `browserfy`:

```shell
gulp
```

Build bundled theme as zip file::

```shell
gulp build
```
